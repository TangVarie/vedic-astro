## 一、DK行星配偶原型

> DK = Chara Karaka中度数最低的星（7K方案）。它代表"灵魂层的配偶指示"。
> DK不预测外貌——它揭示的是配偶的**核心人格基调**和**业力功课**。

### Sun DK
```
配偶核心特质：
  权威感强、有领导力、自尊心高、需要被认可
  可能是管理者、公务员、独立事业者、或"家里的决策者"
  
配偶阳面：自信、有方向感、负责任、"你跟着他/她不会迷路"
配偶阴面：控制欲、ego冲突、不愿妥协、"他/她说了算"

婚姻业力主题：学习"平等vs权威"的平衡
  盘主需要一个强大的伴侣来激发自己，但也必须保持自我空间

Sun DK 入旺(Aries) → 配偶极度自主、领导力炸裂
Sun DK 落陷(Libra) → 配偶想当权威但内在犹豫——表面强势内心纠结
Sun DK 燃烧 → 不适用（Sun不被燃烧）
Sun DK Vargottama → 配偶表里如一的领导者型

⚠️ 性别差异：
  男盘Sun DK → 妻子可能比较强势、有事业心
  女盘Sun DK → 丈夫有权威但可能让你"喘不过气"
```

### Moon DK
```
配偶核心特质：
  情感丰富、关怀型、敏感、重视家庭和安全感
  可能从事照护、教育、餐饮、心理相关工作

配偶阳面：温柔、体贴、直觉强、"你难过的时候他/她第一个感觉到"
配偶阴面：情绪化、过度依赖、被动攻击、"不说但你应该知道"

婚姻业力主题：学习"滋养vs窒息"的平衡
  关系的基调是情感联结——深度很够，但容易被情绪绑架

Moon DK 入旺(Taurus) → 配偶稳定、物质感强、居家型
Moon DK 落陷(Scorpio) → 配偶情绪深沉复杂、占有欲强、但洞察力极强
Moon DK Vargottama → 配偶情绪极度稳定的滋养者型
```

### Mars DK
```
配偶核心特质：
  行动力强、直接、有冲劲、可能脾气大但来得快去得也快
  可能从事工程、运动、军警、外科、创业等高强度领域

配偶阳面：勇敢、保护欲强、执行力强、"说干就干"
配偶阴面：冲动、易怒、缺耐心、"吵架从不认输"

婚姻业力主题：学习"激情vs破坏力"的管理
  关系热度高但摩擦也大——需要有建设性处理冲突的能力

Mars DK 入旺(Capricorn) → 配偶极度有纪律的行动派
Mars DK 落陷(Cancer) → 配偶外表强悍内心敏感——"嘴硬心软"
Mars DK 逆行 → 配偶行动力内化——想法很多但未必外显

⚠️ Kuja Dosha联动：如果Mars DK同时触发Kuja Dosha（Mars在1/2/4/7/8/12宫），
   婚姻冲突概率上升——但不等于婚姻失败，等于"需要更多磨合"
```

### Mercury DK
```
配偶核心特质：
  聪明、善沟通、适应力强、可能比较年轻或显年轻
  可能从事IT、媒体、商业、教育、写作、翻译

配偶阳面：机灵、幽默、好学、"跟他/她聊天永远不无聊"
配偶阴面：善变、理性过度缺温度、话太多、"什么都分析"

婚姻业力主题：学习"沟通vs真正的连接"
  关系建立在智识交流上——但需要注意"光说不做"的陷阱

Mercury DK 入旺(Virgo) → 配偶超强分析能力、追求完美
Mercury DK 落陷(Pisces) → 配偶想法多但表达混乱——"你猜我什么意思"
Mercury DK 燃烧 → ⚠️ 配偶的声音/想法被更强势的人压制——可能在关系中"说不上话"
Mercury DK Vargottama → 配偶极其可靠的沟通者

⚠️ Mercury是中性星——受合相行星影响大。Mercury DK + 与Mars合=配偶爱吵；
   Mercury DK + 与Venus合=配偶有艺术品味且善于表达爱
```

### Jupiter DK
```
配偶核心特质：
  智慧、大气、有教养、可能年长或"像老师"
  可能从事教育、法律、金融、宗教/灵性、咨询

配偶阳面：包容、有远见、值得信赖、"跟他/她在一起觉得人生有方向"
配偶阴面：说教、过度乐观、缺乏实操细节、"道理都对但不落地"

婚姻业力主题：学习"指引vs控制"的区别
  配偶是"导师型伴侣"——但需要平等对话不是单方面教导

Jupiter DK 入旺(Cancer) → 配偶极度滋养的智者
Jupiter DK 落陷(Capricorn) → 配偶务实但缺少灵性温度——"只看结果"
Jupiter DK Vargottama → 配偶深层品质极好——"嫁/娶对了人"（经典吉配）

⚠️ 女盘特别关注：Jupiter是女盘的"丈夫Karaka"（天然象征星），
   当Jupiter同时是DK时=双重确认，丈夫特质非常鲜明
```

### Venus DK
```
配偶核心特质：
  有魅力、注重美感、享受生活、社交能力强
  可能从事艺术、设计、时尚、奢侈品、娱乐、外交

配偶阳面：浪漫、品位好、善于营造氛围、"跟他/她在一起生活有质感"
配偶阴面：物质主义、虚荣、感官沉溺、"花钱没底线"

婚姻业力主题：学习"爱vs执着"的区分
  关系有强烈的感官/美学吸引——但需要超越表面看到灵魂

Venus DK 入旺(Pisces) → 配偶极度浪漫的灵性美人——"爱到骨子里"
Venus DK 入庙(Taurus/Libra) → 配偶稳定的品味型——婚姻品质高
Venus DK 落陷(Virgo) → 配偶爱你但表达挑剔——"用批评表达爱"
Venus DK 燃烧 → ⚠️ 关系中的浪漫/性吸引力被压制——"灵魂说爱但身体拘谨"

⚠️ 男盘特别关注：Venus是男盘的"妻子Karaka"（天然象征星），
   当Venus同时是DK时=双重确认

### Saturn DK
```
配偶核心特质：
  成熟、稳重、有耐力、可能年长或"显老"——不是外貌而是气质
  可能从事制造、政府、法律、审计、长线行业

配偶阳面：可靠、有承担力、长线型伴侣、"嫁/娶给了安全感"
配偶阴面：冷淡、控制、沉闷、过度严肃、"跟他/她在一起有压力"

婚姻业力主题：学习"责任vs自由"的张力
  婚姻是"长征型"——起步慢但后劲足。典型晚婚指标之一

Saturn DK 入旺(Libra) → 配偶纪律+公平兼备——"最佳长线合伙人"
Saturn DK 入庙(Capricorn/Aquarius) → 配偶极度负责任
Saturn DK 落陷(Aries) → 配偶想负责但冲动——"嘴上说慢慢来但行动很急"
Saturn DK 逆行 → 婚姻可能有"延迟重启"模式——先分后合或前缘再续

⚠️ Saturn DK 强烈倾向于"婚姻不会来得早"——30+ 婚是常态，不是问题
```

婚姻业力主题：学习"欲望vs真实"的辨别

⚠️ Rahu DK 在 7K 方案中不出现——若 7K vs 8K 对照表显示 DK 不同（即 8K DK = Rahu），

## UL 计算复习（reader 已提取，如缺失需补算）

```
UL / Upapada Lagna = 第 12 宫的 Arudha Pada（A12 / Vyaya Pada）

⚠️ 注意：UL 是从第 12 宫主推出来的，不是从第 7 宫主。
   把 L7 公式套到 UL 上是常见错误，那样算出来的是 A7 / Darapada（配偶的 Arudha），
   不是 UL。

标准计算步骤：
1. 找出第 12 宫（从 Lagna 起的第 12 宫）
2. 找出第 12 宫主所在星座
3. 从第 12 宫数到第 12 宫主所在星座，得到距离 N（含起点终点）
4. 从第 12 宫主所在星座再向前数 N 个星座 → 初步 UL
5. Arudha 例外：若初步 UL 落在第 12 宫本身，或第 12 宫的对宫（从 Lagna 看的
   第 12 宫 / 第 6 宫），则从该初步点再数第 10 宫作为最终 UL
6. UL 主 = UL 所在星座的统治行星
7. 若与 JHora 输出不一致，以软件结果为准，并记录所用流派

例：Lagna = Aries，12 宫 = Pisces，12 宫主 Jupiter 在 Cancer
   Pisces → Cancer 距离 = 5（Pisces, Aries, Taurus, Gemini, Cancer，含两端）
   从 Cancer 再前进 5 步 = Cancer, Leo, Virgo, Libra, Scorpio → Scorpio
   初步 UL = Scorpio
   Scorpio 既不是 12 宫(Pisces) 也不是 12 宫对宫(Virgo)，例外不触发
   → 最终 UL = Scorpio，UL 主 = Mars

⚠️ 区分两个 Pada：
   - UL = A12 = 婚姻 / 法律落地点 (Upapada)
   - A7 = Darapada = 配偶在世界中的呈现（与 UL 完全不同，请勿混用）
```
```
