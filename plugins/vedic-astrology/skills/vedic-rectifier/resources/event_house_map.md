## 匹配判定规则

### 强匹配 (✅✅)
```
大运/小运主星 = 事件对应宫位的宫主
→ "Venus大运期间结婚，Venus恰好是7宫主" = 强匹配
```

### 中匹配 (✅)
```
大运/小运主星落在事件对应宫位
→ "Jupiter大运期间结婚，Jupiter落在7宫" = 中匹配

大运/小运主星照射事件对应宫位
→ "Jupiter大运期间结婚，Jupiter从1宫照射7宫" = 中匹配

大运/小运主星 = 事件的Karaka
→ "Venus大运期间结婚，Venus是自然婚姻Karaka" = 中匹配
```

### 弱匹配 (⚠️)
```
大运/小运主星与事件宫位的宫主有相位
→ 间接关联，可以接受但权重低
```

### 不匹配 (❌)
```
大运/小运主星与事件宫位无任何关联
→ 强烈暗示时间可能偏差（宫主表错了）
```

## 逆推逻辑

当事件不匹配时，用以下逻辑推理时间偏移方向：

```
已知: 
  事件=结婚, 大运主星=Mars
  当前: Mars管L3和L8 → 与7宫无关 → ❌

逆推:
  Mars管什么星座？→ Aries和Scorpio
  哪个Lagna能让Mars管7宫？
    → Lagna=Aries → 7宫=Libra (Venus管) → Mars不管7宫
    → Lagna=Taurus → 7宫=Scorpio → Mars管7宫 ✅
    → Lagna=Libra → 7宫=Aries → Mars管7宫 ✅
  
  如果当前Lagna=Gemini → 需要向Taurus方向移动
  → 时间要往回调（Lagna度数减小）
```

## 物理标记 / 身体特征验证（⚠️ 已弃用）

> ⚠️ 本节自 v2 起整体弃用，与 vedic-reader 验前事的"永久禁止身体标记"对齐。
>
> 历史背景：早期版本曾使用 Mars 落宫 → 伤疤位置作为校准辅助证据。
> 弃用原因（基于内部经验观察，非实证回测）：
>   - 内部经验显示，身体标记 / 伤疤 / 外貌作证据的稳定性显著低于
>     结构性事实（父母 / 学历 / 搬迁 / 重大事件年份等）
>   - 疾病预测在临床中几乎从未稳定命中
>   - LLM 主动询问身体伤疤会引起用户反感和不信任
>   - Mars 落宫的伤疤映射在临床中可靠性不足（受种族、生活史、运动史
>     等大量非命理因素干扰）
>
> ⚠️ 上述措辞是内部经验权重，不是实证样本回测。若未来积累了真实样本
> 统计，可在 docs/ 单独发布回测报告；在那之前不写具体百分比。
>
> v2 校准协议：
>   - rectifier Step 3a 和 Step 5 盘外验证均使用结构性事实
>     （父母关系、学历水平、搬迁经历、兄弟姐妹数量、出国经历、子女状况、
>      重要事件的 Dasha 对应等）
>   - 禁止追加身体标记、疾病诊断、外貌特征作为校准证据
>   - 与 reader 验前事的【永久禁止】列表完全统一

如果在历史项目中你看到过基于"身体标记"做的校准证据，请理解：
那是旧版本逻辑，新版统一以结构性事实为准。

## 客观可验证事实清单（v2 推荐使用）

> 这些是 rectifier 推荐用于 Step 3a 初步匹配和 Step 5 盘外验证的事实类型。
> 全部为"用户秒答准/不准"的结构性问题，避免主观/性格/身体描述。

| 星盘因素 | 推断 | 验证问题 |
|---------|------|---------|
| 9宫主受Saturn/Rahu严重影响 | 父亲角色缺失/关系疏远 | "您与父亲的关系如何？是否有距离感？" |
| 4宫主受克 + Rahu影响 | 早年家庭不稳/多次搬迁 | "您小时候是否经历过搬家或家庭变动？" |
| 12宫主强+Rahu关联 / 4宫主入12 | 有出国/长期离家经历 | "您是否有长期离开家乡或出国的经历？" |
| 3宫主+3宫行星组合 | 兄弟姐妹数量倾向 | "您有几个兄弟姐妹？" (中国 1979-2015 出生者权重降级) |
| L5 强 / 5宫吉星 | 学历不低/学习能力强 | "您学历或学习经历如何？" |
| L10 落宫 + AmK 状态 | 事业的大致领域 | "您的职业属于以下哪类？技术/管理/创意/服务/其他" |
| L7 + DK 状态 | 婚姻状态/伴侣类型 | "您目前的感情状态？" |

### 使用规则

1. **优先使用此清单**：rectifier Step 3a 和 Step 5 优先用上述类型
2. **覆盖配比**：盘外验证 5 条应至少包含 2 条结构性事实 + 2 条时间锚定事件 + 1 条其他客观事实
3. **措辞**：使用"您是否""您的...如何"等开放问题，让用户自然回答
4. **禁止**：不问身体伤疤、不预测疾病、不描述外貌、不评论性格
