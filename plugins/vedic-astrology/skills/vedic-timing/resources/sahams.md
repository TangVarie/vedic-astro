# Sahams（塔吉卡特殊点）参考

> 来源：Neelakantha《Tajaka Neelakanthi》+ Bhattotpala 注释 + 现代修正
> 用途：vedic-timing Mode C (年盘) Step 3 引用

---

## Sahams 是什么

```
Sahams = Tajaka 体系的"特殊点"——通过计算多颗行星位置得到的虚点。
每个 Saham 指示一个具体生活主题在本年的状态。
（相当于西占的 Arabic Parts / Lots，但传入印度后有所演变。）

经典文献列出约 50 个 Sahams，本 skill 详列 16 个最常用的。
其他更稀有的 Sahams 仅在用户有特定问题时调用。
```

---

## Sahams 通用公式结构

```
所有 Sahams 的通用公式：
  Saham 经度 = (A + B - C) 的归一化结果 (mod 360°)

其中 A、B、C 是三个变量（行星、Lagna、宫位主等的经度）。

昼夜规则（极重要）：
  许多 Sahams 在白天和夜晚出生有不同公式
  白天 = 出生时 Sun 在 7-12 宫（地平线以上）
  夜晚 = 出生时 Sun 在 1-6 宫（地平线以下）
  
  对于年盘 Sahams：使用年盘起算时刻判定昼夜。

归一化：
  若 A + B - C < 0 → 加 360
  若 A + B - C ≥ 360 → 减 360（直到在 0-360 范围内）

Saham 落入星座的宫位 = 该主题在本年的"舞台"
Saham 所在星座的主 = 该主题的"操盘手"
```

## 16 个核心 Sahams

### 1. Punya Saham（功德点）⭐ 必查

```
含义：本年福报、贵人、好运、灵性收获

公式：
  白天：Moon - Sun + Lagna
  夜晚：Sun - Moon + Lagna

解读：
  Punya 落入吉宫(1/4/5/7/9/10/11) → 本年福报显著
  Punya 落入凶宫(6/8/12) → 福报受阻或来自非常规渠道
  Punya 主在年盘强位 → 福报具体兑现
  Punya 主弱 → 福报潜在但需主动激活
  
  Punya Saham 与 Lagna 同宫 → 个人福报年
  Punya Saham 与 10 宫同 → 福报通过事业兑现
  Punya Saham 与 5 宫同 → 福报通过创造/孩子兑现
```

### 2. Yashas Saham（声誉点）⭐ 常用

```
含义：本年声誉、被看见、社会认可

公式：
  日夜相同：Jupiter - Sun + Lagna

解读：
  Yashas 落 10/11/1 → 本年声誉显著
  Yashas 落 12/8 → 声誉受损或来自隐秘渠道
  Yashas 主强 + 在吉位 → 主动获得名声
  
  注意：Yashas 强不一定是好名声——
  若主受凶星影响，可能是"被看见但内容不佳"
```

### 3. Vivaha Saham（婚姻点）⭐ 必查

```
含义：本年婚姻/伴侣事件可能性

公式：
  日夜相同：Venus - Saturn + Lagna

解读：
  Vivaha 落 7 宫 → 本年婚姻强信号
  Vivaha 落 5/2/11 → 恋爱、订婚、伴侣关系积极
  Vivaha 落 6/12/8 → 关系挑战或秘密关系
  
  Vivaha 主与 Lagna 主形成 Ithasala → 关系事件本年发生
  Vivaha 主与 7 宫主形成 Ithasala → 婚姻事件本年发生
  
  ⚠️ 注意：单一 Saham 不足以预测婚姻——
  需与本盘 DK + Vimshottari 7L 大运 + Jupiter 行运 7 宫 综合判断。
```

### 4. Putra Saham（子女/创造点）⭐ 常用

```
含义：本年生育、创造作品、深度教导

公式：
  白天：Jupiter - Moon + Lagna
  夜晚：Moon - Jupiter + Lagna

解读：
  Putra 落 5 宫（自然子女宫）→ 强生育/创造信号
  Putra 落 11/2 → 通过子女/作品获利
  Putra 落 6/8/12 → 创造/生育受阻或代价大
  
  Putra 主与 Jupiter 形成 Ithasala → 创造性事件本年发生
```

### 5. Karma Saham（行动点）⭐ 必查

```
含义：本年事业、行动力、世俗成就

公式：
  日夜相同：Mars - Sun + Lagna

解读：
  Karma 落 10/6/3 → 本年行动力强
  Karma 落 1/11 → 行动有回报
  Karma 落 8/12 → 行动受阻或隐藏
  
  Karma 主强且与 Year Lord 协作 → 事业突破年
```

### 6. Mrityu Saham（危机点）⚠️ 必查

```
含义：本年危机、健康警示、重大转化
（注：不字面翻译为"死亡点"——是"重大转化点"）

公式：
  日夜相同：(8 宫主 - Moon) + Lagna

解读：
  Mrityu 落 8/12 宫 + Mrityu 主弱 → ⚠️ 本年健康/危机警示
  Mrityu 落 6 宫 + 8 宫主参与 Khallasara → ⚠️ 强警示
  Mrityu 落 吉宫 → 危机转化为成长机会
  
  ⚠️ 措辞规则：
    绝不预测"死亡"——只说"需特别关注健康/防御危机"
    "本年是需要特别照顾身体的一年，不要忽视小病信号"
    "做体检、买保险、避免极限运动" 是可以建议的具体动作
    
  与其他 Saham 交叉：
    Mrityu + Roga 同时凶 → 双重健康警示
    Mrityu + Punya 同时强 → 危机伴随福报，转化机会
```

### 7. Vidya Saham（学习点）

```
含义：本年深造、读研、深度学习

公式：
  白天：(Jupiter - Lagna 主) + Lagna
  夜晚：(Moon - Mercury) + Lagna

解读：
  Vidya 落 5/9/2 → 本年学习突破
  Vidya 主强 + 在吉位 → 深造、考试、留学吉
  
  与 Yashas 协同 → 学术 + 声誉双丰收
```

### 8. Vyapara Saham（商业点）

```
含义：本年商业、投资、贸易

公式：
  日夜相同：(2 宫主 - 7 宫主) + Lagna

解读：
  Vyapara 落 2/11 → 商业获利
  Vyapara 落 6 → 通过竞争获利
  Vyapara 落 8/12 → 投资风险或隐性
  
  Vyapara 主与 Mercury 形成 Ithasala → 商业事件本年发生
```

### 9. Roga Saham（疾病点）⚠️

```
含义：本年慢性健康挑战

公式：
  日夜相同：Lagna - Moon + Saturn

解读：
  Roga 落 6/8/12 + 主弱 → ⚠️本年慢性健康挑战
  Roga 落吉宫 + 主强 → 健康改善年
  
  与 Mrityu 同时凶 → 双重警示
  与 Mrityu 反向（一吉一凶）→ "有挑战但可应对"
  
  ⚠️ 措辞同 Mrityu：不灾难化，给具体防御建议。
```

### 10. Bandhu Saham（亲属点）

```
含义：本年亲属、家族事件

公式：
  日夜相同：(Mars - Mercury) + Lagna

解读：
  Bandhu 落 4 宫 → 与近亲深度互动年
  Bandhu 落 8/12 → 家族秘密、隐性事件
```

### 11. Pitru Saham（父亲点）

```
含义：本年父亲事件、与父亲的关系

公式：
  白天：(Saturn - Sun) + Lagna
  夜晚：(Sun - Saturn) + Lagna

解读：
  Pitru 落 9（父亲宫）→ 父亲核心年
  Pitru 落 8/12 → 父亲健康警示
  Pitru 强 → 与父亲关系改善
```

### 12. Matru Saham（母亲点）

```
含义：本年母亲事件、与母亲的关系

公式：
  白天：(Moon - Venus) + Lagna
  夜晚：(Venus - Moon) + Lagna

解读：
  Matru 落 4（母亲宫）→ 母亲核心年
  Matru 落 8/12 → 母亲健康警示
```

### 13. Shoka Saham（悲伤点）

```
含义：本年情感波动、丧失感

公式：
  日夜相同：(Lagna 主 - Saturn) + Lagna

解读：
  Shoka 落 4/12 → 心理低落年
  Shoka 主弱 + 凶位 → 抑郁/丧失警示
  
  与 Mrityu/Roga 同时凶 → 综合心理健康警示
```

### 14. Jadya Saham（停滞点）

```
含义：本年发展停滞、僵局

公式：
  日夜相同：(Saturn - Mars) + Lagna

解读：
  Jadya 强且凶位 → 本年事业/感情/财务停滞
  化解：等待 Jadya 主进入更强 Dasha 期
```

### 15. Mitra Saham（朋友点）

```
含义：本年朋友、社交网络、合作

公式：
  日夜相同：(Saturn - Mars) + Jupiter

解读：
  Mitra 落 11 → 社交收获年
  Mitra 落 6/12 → 朋友冲突或秘密合作
  
  注意：与 Jadya 公式只差最后一项（Lagna vs Jupiter）
```

### 16. Paradesh Saham（异乡点）

```
含义：本年海外、长途、移居

公式：
  日夜相同：(9 宫主 - 9 宫宫首) + Lagna

解读：
  Paradesh 落 9/12 + 主强 → 海外/远行机会
  Paradesh 落 9/12 + 主弱 → 海外计划受阻
  
  与 Vimshottari Rahu/Ketu 大运叠加 → 移民信号
```

## Sahams 综合分析流程

```
Step 1: 计算 4 个必查 Sahams
  Punya (福报)
  Karma (事业)
  Vivaha (婚姻)
  Mrityu (危机)
  
Step 2: 根据 Muntha 位置和 Year Lord 性质，补充 4-6 个相关 Sahams
  Muntha in 4 宫 → 加查 Matru, Bandhu
  Muntha in 5 宫 → 加查 Putra, Vidya
  Muntha in 7 宫 → 加查 Vivaha (已查)、Mitra
  Muntha in 8 宫 → 加查 Mrityu (已查)、Roga、Shoka
  Muntha in 9 宫 → 加查 Pitru, Vidya, Paradesh
  Muntha in 10 宫 → 加查 Karma (已查)、Yashas
  Muntha in 11 宫 → 加查 Punya (已查)、Mitra、Vyapara
  Year Lord = Saturn → 加查 Jadya, Pitru
  Year Lord = Venus → 加查 Vivaha (已查)、Yashas
  Year Lord = Jupiter → 加查 Vidya, Putra, Paradesh
  
Step 3: 对每个 Saham 做"宫位 + 主状态"双层判定
  Saham 落入吉宫 + 主在年盘强位 = 强吉
  Saham 落入凶宫 + 主弱或受困 = 强凶
  其他组合 = 中性或混合
  
Step 4: 找出"叠加信号"
  多个 Saham 都指向同一主题 → 该主题是本年焦点
  例：Punya 落 5 + Putra 落 11 + Vivaha 落 5 → 本年是"恋爱/生育/创造"主导年
```

## 与 Vimshottari 的交叉

```
Saham 信号需要 Vimshottari Dasha 的"激活"才能完全兑现：

  强 Saham 信号 + Vimshottari 相应 MD/AD 激活 → 事件大概率发生
  
  例：Vivaha Saham 落 7 宫 + 主强
      + Vimshottari 当前是 Venus MD / 7L AD
      → 本年是婚姻事件的高置信度窗口
  
  强 Saham 信号 + Vimshottari 不相关 → 信号潜在但本年不一定兑现
  
  例：Karma Saham 落 10 + 主强
      但 Vimshottari 是 12L MD（损耗主题）
      → 事业突破信号存在但本年可能被损耗主题压制
      → 下一年若 Vimshottari 切换到 10L 相关，事件可能"延后兑现"
```

## 写作风格指引

```
✅ 好的写法：
"你今年的 Punya Saham（福报点）落在年盘 11 宫，
而且主星 Venus 在年盘强位（Libra 入庙）。
这是本年最强的'好运信号'之一——
本年通过社交、朋友、目标实现而到来的好运会显著高于平均。

具体怎么用：今年别一个人闷头干。
你的福报今年是'通过他人到达你'的——
多见人、多参与群体活动、接受邀请，
你会发现机会就藏在那些'本来不想去'的场合里。"

❌ 坏的写法：
"Punya Saham at 11th house lord Venus exalted in Libra,
benefic Ithasala with Jupiter indicates auspicious karmic events..."
（参数化、不解释，禁止）
```
