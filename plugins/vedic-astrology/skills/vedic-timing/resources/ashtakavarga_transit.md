## 为什么这是行运分析的"裁判"

普通 Gochara 只看"行星在 Moon 几宫"，但忽略一个关键事实：

**同样的 Saturn 行运到同一个星座，对不同人效果可能完全相反。**

差异从哪来？从本盘 Ashtakavarga 的 bindu 数。
- 你本盘 Saturn 的 BAV 中，Saturn 当前所在星座有 8 个 bindu → Saturn 在你这里"如鱼得水"，吉效率最大化
- 你本盘 Saturn 的 BAV 中，Saturn 当前所在星座有 0 个 bindu → Saturn 在你这里"四面楚歌"，凶效率最大化

这就是为什么同年同月同 Saturn 行运，有人升职、有人失业。

## BAV 解读规则

```
BAV (Bhinnashtakavarga) = 单星阿斯特卡瓦尔加。
每颗行星对应自己的一张 12 星座表，每个星座有 0-8 个 bindu。
全表总和（行总和）固定为：
  Sun: 48 | Moon: 49 | Mars: 39 | Mercury: 54
  Jupiter: 56 | Venus: 52 | Saturn: 39

每颗行星行运到某星座时，看自己 BAV 中此星座的 bindu：

8 bindu = 极强吉位（罕见）
6-7 bindu = 强吉位
5 bindu = 中吉位
4 bindu = 中性
3 bindu = 中凶位
1-2 bindu = 强凶位
0 bindu = 极强凶位（罕见）
```

---

## SAV 解读规则

```
SAV (Sarvashtakavarga) = 全体阿斯特卡瓦尔加。
12 星座每个的 bindu = 该星座所有 7 颗行星 BAV bindu 之和。
全表总和固定为 337。
平均每星座 = 28 bindu。

SAV ≥ 32 = 强场，所有行星到此都偏吉
SAV 28-31 = 中等偏强
SAV 25-27 = 中等
SAV 22-24 = 偏弱
SAV ≤ 21 = 弱场，所有行星到此都偏凶
```

```
                BAV高(≥5)    BAV中(=4)    BAV低(≤3)
SAV高(≥32)     ⭐瞬间超车     建设性吉位    场强但星弱（机会浪费）
SAV中(25-31)   建设性吉位     标准         星弱场也中等（疲软）
SAV低(≤24)     乱世英雄       场弱星中（坎坷） ⚠️结构性毁灭

⭐ 瞬间超车：该行星/该领域开花的明确窗口
⚠️ 结构性毁灭：该行星/该领域踩雷窗口
```

## 优先级（哪颗星的 BAV 最重要）

```
对于行运分析，慢行星的 BAV 权重远超快行星：

1. Saturn BAV：最重要（Saturn 行运周期长，对生活影响深远）
2. Jupiter BAV：第二重要
3. Sun BAV：第三（年级别周期，对官运/权威很关键）
4. Mars BAV：次要（45 天周期但触发力强）
5. Mercury / Venus BAV：参考（快行星触发器作用）
6. Moon BAV：最不重要（2.5 天周期太快）

Rahu/Ketu 没有标准 BAV（古典体系），分析时参考所在星座的 SAV。
```

## 实用临床法则

```
Saturn 行运到本盘 Saturn BAV ≥ 5 的星座：
  即使在 Moon 6/8/12 凶宫，也常表达为"高压建设性结果"
  适合：承担责任、长期投资、重大签约

Saturn 行运到本盘 Saturn BAV ≤ 2 的星座：
  即使在 Moon 11 宫这种吉位，也可能"该来的好不来"
  
Jupiter 行运到本盘 Jupiter BAV ≥ 6 的星座：
  强烈提示该时段是显著好运的窗口期（高置信度信号），
  尤其在该星座所对应的本盘宫位主题上
  适合：开始新项目、深造、谈判
  
Jupiter 行运到本盘 Jupiter BAV ≤ 3 的星座：
  古典"吉位"也会变得"温吞"
  适合：稳健而非扩张

Sun 行运到本盘 Sun BAV ≥ 5 的星座（每年一次，约 30 天）：
  该月份是当年的"曝光/权威/被赏识"月
  
Sun 行运到本盘 Sun BAV ≤ 3 的星座：
  该月不适合做"被看见"的事情
```

## 与 Sade Sati 的交叉

```
Sade Sati 期间，三个连续星座的 SAV/BAV 决定 7.5 年的"实际烈度"：

理想 Sade Sati（轻量级）：
  Moon 前一宫 SAV ≥ 28 + Moon 同宫 SAV ≥ 28 + Moon 后一宫 SAV ≥ 28
  + Saturn BAV 在这三个星座都 ≥ 4
  → 这种 Sade Sati 体感是"忙但有方向"，结果常超预期

困难 Sade Sati（高烈度）：
  三个星座中任意两个 SAV ≤ 24 + Saturn BAV ≤ 2
  → 这种 Sade Sati 体感是"几年都缓不过来"，需特别关怀
  
混合 Sade Sati：
  三个星座差异大 → 7.5 年内有明显的"难期"和"喘息期"
  按月份切片细看
```

## 与 Bhrigu Bindu 的交叉

```
慢行星合相 Bhrigu Bindu 时，看本盘 SAV 在 Bhrigu Bindu 所在星座：

SAV ≥ 30 时合相 = 事件大概率正向（升职/结婚/置业等）
SAV ≤ 24 时合相 = 事件大概率负向（变故/损失/分离等）

这是判断"Bhrigu Bindu 激活事件是吉是凶"的最佳工具。
```
