## 什么是 Sade Sati

Saturn 通过 Moon 所在星座（Janma Rashi）的前一宫、本宫、后一宫，总共三个星座、约 7.5 年的行运。
是吠陀占星里最重要、最被讨论的行运周期。

每人一生平均经历 2-3 次完整 Sade Sati（约每 27.5-30 年一次）。
人生重大主题切换、身份重塑、关系结束/开始，常在 Sade Sati 或 Saturn 关键行运中被放大。
但 Sade Sati 不是唯一触发器——人生重大事件可能发生在 Sade Sati 之外的任何 Saturn / Jupiter / Rahu 关键行运
或 Vimshottari/Chara Dasha 关键 MD-AD 期。判断一次 Saturn 行运的实际强度，必须结合：
  - 当下 Dasha (Vim/Chara 主运叠加)
  - Ashtakavarga 与 Saturn BAV (本盘 Saturn 在通过宫位的累计点数)
  - Moon 的强弱与位置 (Janma Rashi 的本盘状态)
  - 本盘 Saturn 自身的功能性 (FB/FM/Yogakaraka)、入庙入旺、合相情况

简单地说："Sade Sati 期间事情会大"是一个倾向，不是铁律；
反过来"重大事件一定在 Sade Sati 里"更加不成立——这是把基础概率放大成必然的错误推断。

## 三阶段表达

### Aroha（升起期，Saturn 在 Moon 12 宫）

```
主题：损耗、隐性消耗、铺垫
经典表达：
  - 健康开始变差但不严重
  - 财务上隐性流出（保险、医疗、海外、灵性消费）
  - 内心开始"为某件事做准备"，但说不清是什么
  - 离别铺垫：可能将失去的人/事开始浮现端倪
  - 灵性兴趣升高，开始读书/冥想/接触某种修行
  - 睡眠质量下降

身份切换信号：
  开始觉得"过去的自己已经不适用了"，但还没找到新身份。

实操：
  - 主动减少不必要支出，重要事项做保险/备份
  - 体检要做，不要拖
  - 不要在 Aroha 接近结束时（最后 6 个月）做重大投资/换工作
```

### Madhya（顶点期，Saturn 在 Moon 本宫）

```
主题：身份重塑、人生主线大调整
经典表达：
  - 身体出明显问题或显著消瘦/发胖
  - 性格变化别人都看得出来：变沉、变严肃、变内向
  - 主线变化：工作/婚姻/居住地有一项发生根本性调整
  - 旧关系不少结束、新关系暂未稳定
  - 自我认知重写："我究竟想做什么人？"
  - 失眠/抑郁倾向最重的阶段

KN Rao 的临床观察：
  顶点期前 6 个月 + 顶点期 + 顶点期后 6 个月 = 18 个月"高烈度核心"。
  这段时间做的决定，会定义此后 10-20 年的人生路径。

化解原则：
  - 顶点期不抗争，主动调整。逆水推舟 = 加倍痛苦。
  - 该结束的关系结束，该转弯的人生转弯。
  - 健康投资 ≥ 财务投资。
  - 多与长辈、僧侣、命理师/心理咨询师对话。
```

### Avaroha（下降期，Saturn 在 Moon 2 宫）

```
主题：经济、家庭、言语调整
经典表达：
  - 经济压力浮上来（顶点期播种的果实在这里结账）
  - 家庭事务：父母健康、遗产/家产分配、家庭责任加重
  - 言语方式变化：变得更直接/更沉默/更谨慎
  - 财富层面的"清账"：要么显著积累，要么显著流出
  - 顶点期决定的新身份在这里逐步落地

实操：
  - 经济上保守、不投机
  - 主动关心父母健康
  - 不要在 Avaroha 期间打官司或与家人决裂
  - 多吃慢、说慢、做慢
```

## Ardha Sade Sati（半七年半）

```
Saturn 行运到 Moon 的 4 宫 或 8 宫，各持续约 2.5 年。
不算完整 Sade Sati，但有类似的中等强度压力：

Saturn 在 Moon 4 宫（"Kantaka Shani"）：
  家庭、不动产、母亲、心理稳定感受冲击
  约每 15 年一次（与 Sade Sati 错峰）

Saturn 在 Moon 8 宫（"Ashtama Shani"）：
  健康、危机、隐藏问题、财务转化
  古典视为"次级 Sade Sati"，重视程度仅次于完整 Sade Sati
```

## 本盘 Saturn 状态决定 Sade Sati 表达

**这是混派分析的核心。同样的 Sade Sati，不同人体感差异巨大，关键看本盘 Saturn 状态：**

```
Saturn 是 Yogakaraka（金牛 / 天秤 Lagna）：
  → Sade Sati = "极度自律的高产出期"
  → 痛苦真实，但收获实在
  → 这两个 Lagna 的人最该"主动迎接"Sade Sati

Saturn 入旺（天秤座）或入庙（摩羯/水瓶）：
  → Sade Sati 是"硬通货积累期"
  → 过程苦，回报兑现度高

Saturn 落陷（白羊座）：
  → Sade Sati 风险最高
  → 容易陷入"用错力"的循环
  → 强烈建议有命理顾问/心理咨询陪伴

Saturn 是 Maraka（特定宫主，2L或7L）：
  → Sade Sati 期间健康、亲属/配偶健康需重点关注
  → 体检/保险加密

Saturn 受其他凶星严重伤害（与 Mars/Rahu/Ketu 紧密合相）：
  → Sade Sati 表达更激烈
  → 但也更快"过去"
  
Saturn 受 Jupiter 吉相位：
  → "护佑型 Sade Sati"
  → 痛苦中有明显成长，结果好
```

## 与 Vimshottari Dasha 叠加效应

```
Saturn MD/AD + Sade Sati = 双重 Saturn 强化
  → 这是该生命中最显著的 Saturn 表达期
  → 强烈提示该时段是重大人生事件窗口（高置信度信号，方向不必然凶）

Jupiter MD/AD + Sade Sati = "缓冲期"
  → Jupiter 期的智慧/导师/扩张能缓解 Saturn 压力
  → 此时 Sade Sati 偏向"系统升级"而非"系统崩溃"

Sun MD/AD + Sade Sati = 矛盾期
  → Sun 是 Saturn 的天然敌人，体感最累
  → 容易感到不被认可、权威受挑战、健康下降
  → 此时不要硬刚

Rahu/Ketu MD + Sade Sati = "外缘剧变期"
  → 经常伴随移民、转行、突变事件
```

## 化解（不迷信、有实操）

```
实操层（实际行为）：
  - 规律作息：固定睡眠时间，固定运动时间
  - 节俭但不吝啬：减少炫耀性消费，但对老人和弱者保持慷慨
  - 责任担当：主动承担一项他人不愿做的责任（不需要大）
  - 服务行为：每月固定时间做志愿服务/帮助他人
  - 体力劳动：园艺、徒步、做饭等"接地气"活动

灵性层（如有信仰背景）：
  - Hanuman Chalisa：传统认为 Hanuman 是 Sade Sati 期间最强护佑
  - Shani Mantra："Om Sham Shanaye Namaha"
  - Shani Stotra：周六诵读
  - 周六禁酒/禁性/简食
  - 黑芝麻油灯供奉
  - 对老人、清洁工、流浪者保持尊重

避免：
  - 投机、赌博、加杠杆
  - 与长辈/权威无谓对抗
  - 忽视身体的早期警告信号
  - 大决策时"硬撑"，要主动求助
  - Sade Sati Madhya 期内换婚姻状态（除非已是非走不可）
```

## 罕见但需警示的极端情况

```
"三重 Saturn"叠加（极少见但需识别）：
  - Saturn Mahadasha 进行中
  - 同时处于 Sade Sati Madhya
  - 同时 Saturn 行运 conjunct Bhrigu Bindu
  → 此时为该生命中最 Saturn 化的窗口
  → 强烈提示该时段是重大主题切换窗口（极高置信度信号）
  → 强烈建议有专业占星师 + 心理支持系统陪伴
  → 不做永久不可逆决定（如自残、绝交、辞职无下家）

  - 用户处于 Sade Sati 但完全无感觉
  → 通常意味着用户在"逃避"，压力被推迟到 Avaroha 后期一次性结算
  → 反而需更警惕，主动审视生活
```

# 三套 Sade Sati 体系（v2 新增）

> 经典定义的 Sade Sati 是基于 Janma Rashi（出生 Moon 星座）的 Saturn 行运。
> 但 KN Rao 临床派和 SJC 派都使用三套 Sade Sati 并行，分别看不同人生维度。

## 三套体系的对比

| 体系 | 基于 | 7.5 年关注的维度 | 优先级 |
|------|------|---------------|--------|
| **Janma Sade Sati**（经典） | Moon 星座 | 情绪/心理/内在重组/家庭 | 主体，必查 |
| **Lagna Sade Sati** | Lagna 星座 | 身体/身份/对外形象/健康 | 次重要，30 岁后明显 |
| **Sun Sade Sati** | Sun 星座 | 父亲/事业/权威/社会地位 | 男性盘特别明显 |

## Lagna Sade Sati（基于 Lagna 的 Saturn 行运）

### 触发条件
```
Saturn 行运通过 Lagna 星座的前一宫（12宫）、本宫、后一宫（2宫），
共三个星座、约 7.5 年。
```

### 三阶段表达（与 Janma 表达不同，主要影响身体和身份）

```
Aroha (12宫从 Lagna)：
  - 身体感觉"使不上力"，疲劳感强
  - 公开形象有挑战或转弱
  - 倾向退到幕后、不愿被看见
  - 财务上的隐性消耗（医疗、咨询、海外）
  - 体重通常上升（"防御性囤积"）

Madhya (Lagna 本宫)：
  - 身体出明确问题（慢性病显化或既有疾病加重）
  - 形象/身份的根本性改写（年龄感、风格、社会角色）
  - 别人对你的看法发生持久性转变
  - 自我认知层面的"我是谁"重写
  - 关键观察：这阶段长出来的"新的自我"会定型 10-20 年

Avaroha (2宫从 Lagna)：
  - 家庭/经济/言语层面落地
  - 顶点期形成的新身份在家庭和经济层面得到验证或挑战
  - 言语方式有持久转变
  - 财富层面的"清账"
```

### 与 Janma Sade Sati 的同步关系

```
Janma 和 Lagna 的 Sade Sati 通常**不重合**（除非 Lagna = Moon 星座）。
但偶尔会出现"双 Sade Sati 叠加"——
当 Saturn 行运到 Janma 和 Lagna 之间的某个星座时，
可能同时处于一套的 Avaroha 和另一套的 Aroha。

诊断要点：
  - 如果用户同时处于 Janma Madhya + Lagna Aroha → 双重压力，需特别照顾
  - 如果 Janma Avaroha + Lagna Madhya → 内心已平复但身体/形象还在重塑
  - 单一 Sade Sati → 按单套规则解读
```

## Sun Sade Sati（基于 Sun 的 Saturn 行运）

### 触发条件
```
Saturn 行运通过 Sun 星座的前一宫、本宫、后一宫，
共三个星座、约 7.5 年。
```

### 影响维度（主要在父亲/事业/权威方面）

```
Aroha (12宫从 Sun)：
  - 父亲健康开始有隐性问题（或父子关系铺垫张力）
  - 事业上"看不见的损耗"——加班/责任/压力增加但收益未到
  - 与权威人物（上司/长辈）的关系开始重新定义
  - 自我表达受压抑感增强

Madhya (Sun 本宫)：
  - 父亲健康/关系可能出现明确变动
  - 事业核心身份重塑——升职/降职/转行/退休
  - 权威感的根本性重写（"我能不能立得住"的灵魂拷问）
  - 男性盘：此阶段常对应"中年危机"
  - 女性盘：此阶段常对应"与父系权威的和解或决裂"

Avaroha (2宫从 Sun)：
  - 父亲事务的最终结算（健康/遗产/责任移交）
  - 事业重塑的成果落地（新身份的经济回报）
  - 与权威关系的稳定（成为新的权威 或 找到与权威共处的方式）
```

### 与 Janma / Lagna Sade Sati 的优先级

```
- 当三套 Sade Sati 任一处于 Madhya 时 → 都需重点关注
- 男性盘：Sun Sade Sati 与 Janma 同等重要
- 女性盘：Janma Sade Sati 通常主导，Sun Sade Sati 次要（但父亲事务期间仍需关注）
- 三套同时不在 Madhya → 用户处于"Saturn 放风期"，是相对自由的窗口
```

## 在 timing Mode A 中如何使用三套

```
Step 1（行运 t1_sade_sati.md）的工作流程：

1. 查出 Janma Rashi (Moon 星座) → 计算 Janma Sade Sati 状态
2. 查出 Lagna 星座 → 计算 Lagna Sade Sati 状态
3. 查出 Sun 星座 → 计算 Sun Sade Sati 状态
4. 输出三套各自的当前阶段（未启动/Aroha/Madhya/Avaroha/已结束）
5. 标注是否存在"双套或三套叠加"
6. 按主导套优先解读：
   - 任一在 Madhya → 重点写该套的影响
   - 多套同时在不同阶段 → 分别写，并标注叠加效应
   - 全部不在期内 → 简短说明，转入其他行运分析

✅ 好的写法：
"你的 Janma Sade Sati 在 2024 年 1 月已结束，但 Lagna Sade Sati 
 在 2025 年 8 月刚进入 Aroha——意味着你心里的风暴已经过去，
 但身体和形象层面的重塑才刚开始。
 你的 Sun Sade Sati 要到 2031 年才启动，所以父亲和事业层面这几年还是平稳的。"

❌ 坏的写法：
（只写 Janma，不查 Lagna 和 Sun，导致用户感觉"我明明很累但你说我没在 Sade Sati"）
```

## 不强推化解

```
古典派会推荐 Sade Sati 期间做 Shani Pooja、戴蓝宝石、念 Hanuman Chalisa 等。
本系统的立场：
  - 提供信息：传统化解方式是什么、为什么古人这么做
  - 不强推：用户有信仰背景可参考，无信仰背景给世俗化建议
  - 世俗化建议：作息规律、定期体检、降低支出、不做重大不可逆决定、寻求心理支持
  - 永远不说"必须做某仪式否则会有灾"——这是恐吓式占星，禁止
```
