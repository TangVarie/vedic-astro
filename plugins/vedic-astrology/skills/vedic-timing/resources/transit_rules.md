# 行运规则参考

> 来源：Parashari 的 Gochara 章节 + KN Rao 临床调整
> 用途：vedic-timing Mode A (行运) Step 2-3 引用

## 总原则

```
吠陀占星的行运分析与西占不同：
  1. 主参考点是 Moon 星座（Janma Rashi），不是 Ascendant
  2. 关注"从 Moon 数第几宫"
  3. 慢行星（Saturn / Jupiter / Rahu / Ketu）效果重要
  4. 快行星（Sun / Mercury / Venus / Mars）效果次要，多作触发器

KN Rao 修正：
  也参考从本盘 Lagna 数的行运（"Lagna-based Gochara"）
  当 Moon-based 和 Lagna-based 同时指向同一信号 = 强化
  当二者冲突 = 看 Moon 还是 Lagna 更"重"（看 Shadbala）
```

## Jupiter 行运（约每星座 12-13 个月）

### 从 Moon 数

```
1宫 (Moon 同宫)：中吉
  - 心情扩张、学习兴趣、新机会出现
  - 但与本盘 Moon 合相会有"过度乐观"风险
  - 体重可能增加

2宫：强吉
  - 财富、家庭、言语方面好运
  - 适合谈判、签约、入新职位

3宫：中吉
  - 兄弟、短途、技能学习有进展
  - 适合开始新项目（小规模）

4宫：凶
  - 家庭、不动产、心理稳定受冲击
  - 容易搬家、装修、与母亲关系紧张
  - 古典视为"Jupiter 退化"

5宫：强吉
  - 创造力、子女、恋爱、教育大利
  - 最适合开始创作、备孕、深造
  - "Guru Bala" 加持，灵性进展

6宫：中吉（古典视为凶位但实际多为吉）
  - 健康改善、争议化解、债务降低
  - 适合启动健身、戒断、还债
  - 现代修正：Jupiter 在 6 宫的"扩张"反而帮助你赢

7宫：强吉
  - 婚姻、合伙、合作大利
  - 单身者高概率遇到长期伴侣
  - 已婚者关系深化或公开化
  - 商业合作签约最佳窗口

8宫：凶
  - 隐性焦虑、健康暗流、突变事件
  - 不适合做重大投资
  - 但"祖辈/继承"主题可能进展
  - 灵性追求深化（神秘学/瑜伽/冥想突破）

9宫：强吉
  - 远方、导师、运气、深度学习大利
  - 出国/出游/拜师/读研的最佳窗口
  - "Guru in Bhagya" — 命运扩张

10宫：凶
  - 事业表面光鲜但有承诺过度风险
  - 古典视为"过度承诺导致负担"
  - 现代修正：高位者反而是"高曝光高责任"窗口
  - 应避免：贸然换工作

11宫：强吉
  - 收入、社交、人脉、目标实现
  - 老朋友重新出现，新朋友加入

12宫：凶（但有灵性益处）
  - 海外、损耗、灵性、隔离
  - 适合做长期闭关、海外项目、艺术创作
  - 不适合：高调社交、大额投资
```

### Vedha（行运对冲，必查）

```
Jupiter 的 Vedha 点：Moon 9 宫
  → 若 Saturn 同时行运到 Moon 9 宫，Jupiter 其他位置的吉相被部分对冲
  
Jupiter 行运 9 宫吉效失效条件：
  → Saturn 行运到 Moon 7 宫时 Jupiter 在 9 宫的好运被锁

最强 Jupiter 窗口：Jupiter 在吉位 + Saturn 不在 Vedha 点 + 该星座 SAV ≥ 30
```

## Saturn 行运（约每星座 2.5 年）

### 从 Moon 数

```
1宫 (Moon 同宫)：Sade Sati Madhya — 见 sade_sati.md
2宫：Sade Sati Avaroha — 见 sade_sati.md
12宫：Sade Sati Aroha — 见 sade_sati.md

3宫：强吉（Saturn 最喜欢的位置之一）
  - "Sadhe Saati 后的复苏期"
  - 努力得到回报、技能精进
  - 兄弟/沟通/勇气方面成长

4宫：Kantaka Shani（半 Sade Sati）
  - 家庭压力、母亲健康、不动产困扰
  - 心理低落、根基不稳

5宫：凶
  - 创造力受阻、与子女关系紧张
  - 恋爱/婚姻拖延
  - 投资保守

6宫：强吉（与 Jupiter 在 6 宫类似，Saturn 也喜欢竞争场）
  - 工作努力得到承认
  - 健康问题逐步治愈
  - 债务清理、对手退让

7宫：凶
  - 婚姻/合伙重大调整
  - 单身者长期单身或遇到"难处理"对象
  - 已婚者关系冷淡或分居

8宫：Ashtama Shani（半 Sade Sati）
  - 隐藏问题爆发
  - 健康危机倾向
  - 灵性突破或重大转化

9宫：凶
  - 运气下滑、长辈健康问题
  - 父亲/导师方面打击
  - 不适合远行/换地

10宫：中吉（古典视为"Saturn 主场"）
  - 事业责任加重但是建设性
  - 长期努力开始有回报
  - 升职/转型窗口
  - KN Rao 视为 Saturn 最重要的行运位之一

11宫：强吉（Saturn 最喜欢的位置）
  - 收入显著增加、目标实现
  - 努力的复利在此结算
  - "Saturn in 11th" 是行运里的金牌位置

12宫：Sade Sati 启动（如前述）
```

### Saturn 的 Vedha 点

```
Saturn 的 Vedha 点：Moon 12 宫
  → 若 Jupiter 同时行运到 Moon 12 宫，Saturn 在其他位置的凶相被部分缓冲
  → 这是 Sade Sati 期间 Jupiter 在 12 宫时反而成"保护伞"的原因
```

## Rahu / Ketu 行运（约每星座 18 个月，逆行）

```
Rahu 和 Ketu 始终对冲 180°，所以分析"轴线"而非单点。
轴线移动一次 = 18 个月一次"人生主题切换"。
每人一生经历约 4-5 次完整轴线切换。

强信号组合（Rahu/Ketu 落在本盘哪两个宫）：

Rahu 1 / Ketu 7：
  → 自我膨胀 / 伴侣解离
  → 适合：自我品牌建设、单飞
  → 风险：婚姻冷淡、合作破裂

Rahu 2 / Ketu 8：
  → 财富野心 / 危机淡化
  → 适合：财富积累、家庭扩张
  → 风险：财务过度承诺

Rahu 3 / Ketu 9：
  → 技能/勇气放大 / 远方/导师疏离
  → 适合：转型尝试、自学新技能
  → 风险：与传统/家族信仰冲突

Rahu 4 / Ketu 10：
  → 家庭/根基放大 / 事业放手
  → 适合：买房、搬迁稳定下来、回家
  → 风险：事业停滞、被边缘化

Rahu 5 / Ketu 11：
  → 创造力/恋爱/孩子放大 / 收入/社交解构
  → 适合：备孕、创作、开始恋爱
  → 风险：投机失败、朋友圈消解

Rahu 6 / Ketu 12：
  → 竞争/健康/工作放大 / 灵性/海外/隔离淡化
  → 适合：硬刚对手、解决长期健康问题
  → 风险：身陷争议、过劳

Rahu 7 / Ketu 1：
  → 伴侣/合作放大 / 自我消解
  → 适合：婚姻、合伙、被引荐
  → 风险：失去自我、过度依赖

Rahu 8 / Ketu 2：
  → 转化/危机/继承放大 / 家庭/财务淡化
  → 适合：神秘学/治疗工作、突变机会
  → 风险：财务危机、家庭事变

Rahu 9 / Ketu 3：
  → 远方/导师/运气放大 / 技能/勇气消解
  → 适合：出国、读研、拜师
  → 风险：缺乏行动力、过度理想化

Rahu 10 / Ketu 4：⚠️ 经典强组合
  → 事业野心↑ / 家庭根基松动
  → 适合：升职、出名、转行
  → 风险：与家人疏远、心理无根

Rahu 11 / Ketu 5：
  → 收入/社交放大 / 创造力/恋爱/孩子淡化
  → 适合：横向拓展、社交资本积累
  → 风险：感情冷淡、子女问题

Rahu 12 / Ketu 6：
  → 海外/灵性/隐居放大 / 工作/健康/争议淡化
  → 适合：移民、长期闭关、艺术
  → 风险：脱离现实、隐性消耗
```

### Rahu/Ketu Return 与 Square

```
Rahu Return（约 18 年一次，更精确是 18.6 年）：
  Rahu 行运回到本盘 Rahu 位置
  → 重大人生议题再演 / 该议题升级
  → 一生约 4-5 次（18/36/55/74/93 岁前后）

Rahu Square（约 9 年一次）：
  Rahu 行运 90° 于本盘 Rahu
  → 中度议题再演
  → 一生约 8-10 次

Nodal Reversal（约 9.3 年一次）：
  Rahu 行运到本盘 Ketu 位置（180° 对冲）
  → 重大方向反转
  → 极常对应人生重大转折
```

## Lagna-based Gochara（KN Rao 补充法）

```
除了从 Moon 数行运效果，也从 Lagna 数。
当 Moon-based 和 Lagna-based 都指向吉位 = "双吉位"，效果最强。
冲突时：

Moon-based 吉 + Lagna-based 凶 = "心情好但环境差"
Moon-based 凶 + Lagna-based 吉 = "外界顺但心累"
两者都吉 = 全面顺
两者都凶 = 重点防御期

KN Rao 经验：长期事件看 Lagna，短期心情看 Moon。
```
