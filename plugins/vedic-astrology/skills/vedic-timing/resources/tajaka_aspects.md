# Tajaka 相位 + 12 Yogas 参考

> 来源：Neelakantha《Tajaka Neelakanthi》+ Yavanacharya 传承
> 用途：vedic-timing Mode C (年盘) Step 4 引用

---

## Tajaka 相位 vs Parashari 相位的根本区别

```
Parashari 相位：
  按"整宫"（whole-sign）计算
  例：Jupiter 在 Aries → 它对所有 Leo, Sagittarius 的行星都有 trine 相位
  不考虑度数差

Tajaka 相位：
  按"度数差"（orb-based）计算
  例：Jupiter 在 Aries 5° → 它只对在 trine 距离 ±x° 内的行星有相位
  必须考虑 orb（容度）
  
所以同一颗星，Parashari 视角和 Tajaka 视角的"相位关系"可能完全不同。
本 skill 在年盘分析中只用 Tajaka 相位，不混用 Parashari。
```

## Orb（容度）表

```
不同行星有不同的 orb（容度）：

| 行星 | Orb (度) |
|------|---------|
| Sun     | 15° |
| Moon    | 12° |
| Mercury | 7° |
| Venus   | 7° |
| Mars    | 8° |
| Jupiter | 9° |
| Saturn  | 9° |

两颗星的"相位 orb" = 两颗星 orb 平均值
例：Jupiter 与 Saturn 的相位 orb = (9 + 9) / 2 = 9°
   Sun 与 Moon 的相位 orb = (15 + 12) / 2 = 13.5°

只有当两星实际度数差 ≤ 相位 orb 时，才算"形成相位"。
```

## Tajaka 相位类型

```
Tajaka 体系承认 5 种主要相位：

1. Conjunction（合相）— 同一星座，度数差 ≤ orb
2. Opposition（180°对冲）— 对面星座，度数差 ≤ orb
3. Trine（120°三合）— 相距 4 个星座，度数差 ≤ orb
4. Square（90°四分）— 相距 3 个星座，度数差 ≤ orb
5. Sextile（60°六合）— 相距 2 个星座，度数差 ≤ orb

每种相位的吉凶倾向：
  Conjunction → 合作/融合（性质取决于两星）
  Opposition → 张力/对峙
  Trine → 和谐/支持
  Square → 摩擦/挑战
  Sextile → 机会/温和
```

## 12 主要 Tajaka Yogas

### 1. Ikkavala（合相格）

```
定义：两颗星在同一星座（不论度数）。
含义：两颗星的能量"共享一个舞台"。
解读：
  两吉星 Ikkavala → 双重吉运
  两凶星 Ikkavala → 双重压力
  一吉一凶 → 主题混合
  
注：Ikkavala 不等于 Tajaka conjunction（需要度数差 ≤ orb）。
   Ikkavala 是"同星座"，Conjunction 是"同星座 + 度数接近"。
```

### 2. Indu-Vara（互旺格）

```
定义：两颗星互相落在对方的入旺星座。
例：A 星入旺 X 星座，B 星入旺 Y 星座；A 在 Y，B 在 X。
含义：两颗星都在"完美对调"，互相赋能。
极强吉 Yoga，少见。
```

### 3. Ithasala（成就格）⭐ 最重要

```
定义：一颗较快的星正在追赶（不断接近）一颗较慢的星，
      且两星度数差 ≤ 相位 orb。

行星速度从快到慢：Moon > Mercury > Venus > Sun > Mars > Jupiter > Saturn
（注：Mercury 和 Venus 速度不固定，需看是否逆行）

类型：
  Vartamana Ithasala（即时成就）：度数差极小（<2°）
    → 影响立刻发生，年盘内事件迅速兑现
  
  Bhavishya Ithasala（未来成就）：度数差较大但仍在 orb 内
    → 影响在本年内逐步发生

含义：两颗星的能量"完成一次对话"，主题被激活。

⭐ 这是 Tajaka 最重要的 Yoga，事件预测的核心。

解读：
  Year Lord 与 Lagna 主形成 Ithasala → 本年主题强力激活
  Vivaha Saham 主与 Venus 形成 Ithasala → 婚姻事件大概率
  Mrityu Saham 主与 8L 形成 Ithasala → ⚠️健康警示激活
  Punya Saham 主与 Jupiter 形成 Ithasala → 福报兑现
```

### 4. Ishraf（解脱格）

```
定义：一颗较快的星正在离开（远离）一颗较慢的星，
      但两星度数差仍 ≤ 相位 orb。

含义：两颗星的能量"刚刚完成对话，正在分开"。
主题已经发生过，正在收尾。

解读：
  如果是吉星 Ishraf → 本年初期有好事，正在落定
  如果是凶星 Ishraf → 本年初期有挑战，正在过去
  
注意：Ishraf 是"事件已发生"的信号，与 Ithasala（"事件将发生"）相反。
```

### 5. Nakta（传话格）

```
定义：两颗慢星不直接形成相位，但通过一颗中间快星传递信息。
      快星先离开 A，再接近 B，形成 A→Quick→B 的传话链。

含义：A 和 B 通过中介间接合作。

解读：常对应"通过他人才能完成的事"——
  例：Vivaha 主 (慢) 与 Venus (慢) 通过 Moon (快) Nakta
  → 婚姻通过他人介绍/帮助而成
```

### 6. Yamya（反传话格）

```
定义：与 Nakta 相反——快星从 A 反向接近 B，信息扭曲传递。
含义：信息在传递中失真，合作出现问题。

解读：常对应"通过中介反而搞砸的事"——
  尽量避免通过第三方处理本年关键事项。
```

### 7. Manau（速会格）

```
定义：两颗快星互相形成 Ithasala。
含义：短期事件的迅速发生。

解读：通常对应本年某月内的具体事件，不是大主题。
  例：Mercury 与 Moon Manau → 本月内有重要沟通/小决策
```

### 8. Kambula（多重成就格）

```
定义：多个 Ithasala 同时存在（≥3 组）。
含义：复合事件密集发生年。

解读：本年是"密集发生事件年"，不太适合静养、适合主动。
  Year Lord 参与 Kambula → 本年特别热闹
```

### 9. Gairi-Kambula（受阻多重格）

```
定义：本应形成 Kambula 但被某颗凶星打断。
含义：复合计划被推迟或受阻。

解读：本年很多事"差一点就成"，但需要补充某个缺失环节。
```

### 10. Khallasara（围困格）⚠️

```
定义：一颗星陷入大量凶相（与多颗凶星形成 square/opposition）。
含义：该主题在本年严重受困。

解读：
  Year Lord 在 Khallasara → ⚠️本年整体困难
  Lagna 主在 Khallasara → ⚠️健康/自我状态警示
  Saham 主在 Khallasara → 该 Saham 对应主题严重受挫
  
⚠️ 与 Mrityu Saham + Roga Saham 同时出现 Khallasara → 强健康警示
```

### 11. Radda（拦截格）⭐ 化解 Yoga

```
定义：一颗吉星拦截两颗凶星的相位，阻止灾难。
含义：吉星化解凶相，主题被保护。

解读：本年原本有的灾难信号被吉星化解。
  Jupiter Radda → 智慧化解
  Venus Radda → 情感/资源化解
  Moon Radda（强 Moon）→ 滋养化解

⭐ Radda 是 Tajaka 最幸运的 Yoga 之一——
  本年即使 Mrityu/Roga 等凶 Saham 信号强，
  若它们的主受 Radda 保护 → 警示降级。
```

### 12. Duphali-Kutha（双凶合相）⚠️

```
定义：两颗凶星紧密合相（在 Tajaka orb 内）。
含义：双重压力，特别在合相所在的宫位。

解读：
  Mars + Saturn 紧密合相 → ⚠️暴力/手术/事故警示
  Saturn + Rahu 紧密合相 → ⚠️突变性损失警示
  Mars + Rahu 紧密合相 → ⚠️冲突/事故警示
  Mars + Ketu 紧密合相 → ⚠️意外/分离警示
  
⚠️ 化解：检查是否有 Radda 拦截。若有 → 警示降级。
```

## Yoga 综合判定流程

```
Step 1: 识别"激活型" Yoga（事件预测的关键）
  扫描所有 Ithasala，按以下优先级排序：
    a) Year Lord 参与的 Ithasala
    b) Lagna 主参与的 Ithasala
    c) Muntha 主参与的 Ithasala
    d) 重要 Saham 主参与的 Ithasala
  
  每一个 Ithasala 都是本年的"事件触发器"，
  必须解读：这两颗星的能量结合 → 什么类型的事件 → 哪个月可能爆发

Step 2: 识别"警示型" Yoga（防御策略的关键）
  扫描：
    a) Khallasara（围困）
    b) Duphali-Kutha（双凶合相）
    c) 重要 Saham 主在凶相
  
  每一个警示信号都需要：
    - 说明主题（健康/财务/感情/事业）
    - 说明强度（弱警示/中警示/强警示）
    - 给具体防御建议
  
  ⚠️ 必查 Radda 是否化解 → 若化解则降级

Step 3: 识别"密集型" Yoga
  Kambula → 本年事件密集
  Gairi-Kambula → 本年事件想做但受阻
  
Step 4: 识别"过渡型" Yoga
  Ishraf → 本年初期有事件正在落定
  这些事件需要追溯到上一年盘
```

## 写作风格指引

```
✅ 好的写法：
"你今年最重要的事件信号是：
Vivaha Saham（婚姻点）的主星 Venus 正在追赶 Jupiter，
两星距离 4°，在 Tajaka 体系里这构成一个 Bhavishya Ithasala。

人话翻译：Venus 是关系，Jupiter 是扩张和承诺。
本年内（最可能在第 7-9 个月之间），
你大概率会面对一个'需要扩展承诺范围的关系事件'——
单身的话可能是确定关系，
恋爱的话可能是订婚或同居，
已婚的话可能是孩子/置业等共同承诺。

不需要刻意安排，让它自己发生。"

❌ 坏的写法：
"Vivaha Saham lord Venus in Bhavishya Ithasala with Jupiter
indicates marital karmic activation per Tajaka Neelakanthi..."
（参数化、术语堆叠、不解释，禁止）
```
