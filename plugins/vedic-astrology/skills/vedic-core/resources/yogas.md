# 格局(Yoga)评估方法论

> 用途：core在十大板块第10项"格局评估"时参考
> 原则：说潜力和突破口，不定命

---

## 主要格局扫描清单

### Dharma-Karma Yoga（使命格局）
```
条件：9宫主和10宫主互动（合相/互视/互溶）
含义：事业有使命感，社会认可度高
评估：
  参与星强 → "你的事业天然带有意义感，容易获得社会尊重"
  参与星弱 → "有这个潜力但需要特定Dasha时期才能激活"
```

### Dhana Yoga（财富格局）
```
条件：2宫主和11宫主互动
      或 2/11宫主和1/5/9宫主互动
含义：财富积累能力强
评估：
  看参与星的Shadbala和尊贵度
  看2宫和11宫的SAV（资源是否充裕）
```

### Raja Yoga（王者格局）
```
条件：三角宫主(1/5/9) + 角宫主(1/4/7/10) 合相或互视
含义：权力/地位/影响力
评估：
  纯度 = 参与星是否同时管凶宫（如果是则打折）
  强度 = 参与星的Shadbala
```

### Viparita Raja Yoga（逆境王格局）
```
条件：6/8/12宫主互溶或合相于6/8/12宫
含义：在困难中崛起，从别人的损失中获益
评估：
  存在 → "你有一种特殊能力：越是困难的环境，你越能找到出路"
  注意：不要过度美化，VRY的代价是必须经历逆境
```

### Gajakesari Yoga（象王格局）
```
条件：Jupiter和Moon互为角宫（相距1/4/7/10宫）
含义：智慧+情感稳定，学习能力强
评估：
  Jupiter和Moon的Shadbala决定实现程度
```

### Chandra-Mangala Yoga（月火格局）
```
条件：Moon和Mars合相或互视
含义：行动力+直觉力结合，适合创业
评估：
  吉利组合但有情绪波动的副作用
```

### Kemadruma Yoga（月亮孤独格局）
```
条件：Moon前后两宫(从Moon算2宫和12宫)均无行星
含义：情绪孤立/心理孤独感，内心缺乏安全感
评估：
  成立 → "你的内心容易感到孤独，需要主动建立情感支持系统"
  
  解除条件（任一满足即解除）：
    - Moon在Kendra(1/4/7/10宫)
    - Moon有行星合相
    - Moon受Jupiter相位
    
  解除 → "Kemadruma被缓解，孤独感减弱但底色仍在"
```

### Pancha Mahapurusha Yoga（五大贤者格局）
```
条件：Mars/Mercury/Jupiter/Venus/Saturn 落在Kendra(1/4/7/10) 且 入旺或入庙
  → 5颗星各形成一个独立格局：

  Ruchaka (Mars in Kendra + 入旺Capricorn/入庙Aries或Scorpio)
    → 勇武、领导力、体魄强健、军事/执法/工程才能

  Bhadra (Mercury in Kendra + 入旺Virgo/入庙Gemini)
    → 智慧、商业才能、沟通/写作/教育天赋

  Hamsa (Jupiter in Kendra + 入旺Cancer/入庙Sagittarius或Pisces)
    → 学识、精神智慧、道德感、教育/学术/法律

  Malavya (Venus in Kendra + 入旺Pisces/入庙Taurus或Libra)
    → 艺术鉴赏、人际魅力、奢华享受、美学/设计/外交

  Shasha (Saturn in Kendra + 入旺Libra/入庙Capricorn或Aquarius)
    → 权威、纪律、管理才能、政治/行政/建筑

评估：
  参与星燃烧/逆行/战争 → 格局打折
  参与星受凶星相位 → 效力减弱
  成立 → "你有[格局名]，在[领域]有先天的高起点"
```

### Sunapha / Anapha / Durudhura（月护三格局）

```
基于 Moon 前后宫的行星配置（Sun 不算）：
  
  Sunapha Yoga：Moon 后一宫（从 Moon 数第 2 宫）有行星
    → "情绪有支撑"——内在感受能落到现实
    → 财富+稳定基调
    
  Anapha Yoga：Moon 前一宫（从 Moon 数第 12 宫）有行星
    → "情绪有铺垫"——感受被酝酿出来
    → 名声+表达力
    
  Durudhura Yoga：Moon 前后两宫（第 12 和第 2）都有行星
    → 双向滋养
    → 富贵+人脉
    
解除条件：
  Sun 在 Moon 前后宫不算（Sun 距 Moon 过近会消解此格）
  
评估：
  - 参与的是吉星 → 该格局成立且偏正面
  - 参与的是凶星 → 格局成立但偏严苛（"靠纪律/危机来推动情绪"）
  - 与 Kemadruma 互斥：有 Sunapha/Anapha/Durudhura → 自动解除 Kemadruma
  
人话翻译：
  Sunapha → "你内心的想法能找到落地的支撑点"
  Anapha  → "你的情绪表达有铺垫和氛围感，容易被人感知"
  Durudhura → "你内在感受系统的前后都有人/事接住"
```

### Adhi Yoga（自然王者格局）

```
条件：从 Moon 起数，第 6/7/8 宫均有"自然吉星"（Mercury/Venus/Jupiter）
  → 不需要参与星互相合相，只要这三个宫位各自被吉星占据
  → 三个宫位可以是同一颗吉星（极罕见），通常是不同吉星
  
含义：
  天生具备"赢家"气质——不依靠权位也能获得他人主动支持
  适合做需要长期人格信任的工作（公职、咨询、教育、医疗）
  
评估：
  - 参与的吉星强（入旺/入庙/Vargottama）→ 全力 Adhi Yoga
  - 参与星弱（落陷/燃烧）→ 格局存在但表达受限
  - Jupiter 参与 → 偏"智慧/道德"型王者气
  - Venus 参与 → 偏"魅力/和谐"型王者气
  - Mercury 参与 → 偏"沟通/适应"型王者气
  
人话翻译：
  "你不用刻意去争，别人会自然信任和支持你。
   关键是别用力过猛——天生光环最怕'我要让大家服我'的姿态。"
```

### Lakshmi Yoga（吉祥/财富格局）

```
条件（任一即成立）：
  1. 9 宫主（L9）落在 Kendra/Trikona 且力量强（Shadbala ≥ 120%）
     且与 Venus 形成合相/三合/对冲
  2. Venus 入旺或入庙 + 9 宫主在 Kendra/Trikona
  
含义：
  财富与好运、美感与丰盛同时具备
  钱来得"优雅"，不是苦力劳动型积累
  容易吸引高品质资源（人脉、机会、伴侣）
  
评估：
  - 参与星燃烧/落陷 → 格局存在但需特定 Dasha 激活
  - L9 是 Lakshmi Yoga 的"灵魂主"，其状态决定上限
  - 注意：Lakshmi Yoga ≠ "不劳而获"，它是"努力的事容易得到回报"
  
人话翻译：
  "你的财富路径偏'美'的——通过优雅的、有审美的、利他的事情赚钱，
   而不是单纯靠拼体力或熬时间。"

兑现窗口：Venus / L9 / L2 的 Dasha 期间最强
```

### Saraswati Yoga（才艺格局）

```
条件：Mercury / Venus / Jupiter **三星共同** 落在 Kendra (1/4/7/10) 或 Trikona (1/5/9)
  且至少有一颗位于自己的 Sign / 入庙 / 入旺
  
含义：
  艺术、学问、表达、创造力的全方位天赋
  适合做需要"知识+审美+教导"三合一的事
  典型职业：教师、艺术家、作家、咨询顾问、设计师
  
评估：
  - 三星全在 Kendra → 偏"实战型才艺"（能赚钱）
  - 三星全在 Trikona → 偏"灵感型才艺"（有创造力但商业化弱）
  - 三星跨 Kendra+Trikona → 最理想，知行合一
  - 有任一星燃烧/落陷 → 格局存在但相应方向（思维/审美/智慧）打折
  
人话翻译：
  "你天生有'又能学又能创又能讲'的复合能力。
   关键是别只盯一项——三项一起用才是这个格局的真正威力。"
```

### Kala Sarpa Dosha（业力轴/蛇毒格）

```
条件：所有 7 颗主行星（Sun~Saturn）全部落在 Rahu-Ketu 轴的一侧
  → 即所有行星都被 Rahu 和 Ketu "包围"在 180° 弧内
  
分类：
  - Kala Sarpa Yoga：所有行星从 Rahu 顺时针到 Ketu（正向包围）
    → "主动型业力"：此生有强烈的方向感和使命感，但路径单一
  - Kala Sarpa Dosha：所有行星从 Ketu 顺时针到 Rahu（反向包围）  
    → "被动型业力"：此生像被命运牵引，意志力相对薄弱
  
含义：
  这是吠陀占星中最有争议的格局之一。
  古典派认为是"凶格"，需要 Remedies 化解（如 Naga Pooja）
  现代派（KN Rao 等）认为是"业力主题集中"，不必然凶
  
本系统的立场：
  - 不强化恐惧叙事（"你有业力魔咒"）
  - 标注为"此生主题高度集中在 Rahu-Ketu 轴指向的领域"
  - 不推荐宗教化解，建议在 Rahu/Ketu 大运期间特别关注主题领域
  
解除条件（任一即解除）：
  - 任何一颗行星落在 Rahu-Ketu 轴线外（即使是 1° 也算解除）
  - Sun + Moon 与轴线呈 trine（120°） → 部分解除
  - 强 Yogakaraka 与轴线无关 → 部分解除
  
人话翻译：
  "你这辈子的主题非常集中，几乎所有能量都围绕一个轴线（Rahu 在 X 宫 → Ketu 在 Y 宫）展开。
   这意味着：你的成就和挑战都在这条线上，分散精力到其他领域反而事倍功半。"

⚠️ 严格禁止：不要说"你被命运诅咒""你前世造孽""必须做法事化解"等吓人话术
```

### Vasumati Yoga（财富格局之二）

```
条件：所有自然吉星（Mercury/Venus/Jupiter，及增强的 Moon）落在 Upachaya 宫（3/6/10/11）
  → 全部都落在"努力越多回报越多"的宫位
  
含义：
  长期努力型财富积累
  钱不是天降，而是通过持续的、积累型的事情赚来
  适合做需要 5-10 年沉淀才见效的事
  典型路径：长期投资、专业能力变现、慢生意、品牌建设
  
评估：
  - 至少 2 颗吉星在 Upachaya → 弱 Vasumati（基础财富稳定）
  - 3 颗以上 → 完整 Vasumati（财富积累上限高）
  - 全部 4 颗（含 Moon）→ 大 Vasumati（财富量级跃升潜力）
  
人话翻译：
  "你的财富是'熬'出来的——3 年起步，5 年见效，10 年成势。
   不要羡慕快速暴富的人，你的盘不支持那种路径，但你的曲线最终会更高。"

兑现窗口：进入 30 岁后陆续显化，Saturn 大运 + Jupiter 流年是关键加速期
```

## 评估框架

```
Step 1: 扫描上述格局是否存在
Step 2: 评估参与星状态

  格局存在 + 参与星强(Shadbala>120%) = 高实现概率
  → "这个格局在你盘中是活跃的，已经或即将体现在生活中"

  格局存在 + 参与星弱(Shadbala<100%) = 低实现但有潜力
  → "这个格局存在但需要适当的时机（通常是参与星的Dasha）才能激活"

  格局存在 + 参与星受损(燃烧/落陷) = 有格局但增益×0.2
  → "有这个倾向但实现起来会遇到阻力"

  格局不存在 = 看功能性等价物
  → 没有正式的Raja Yoga，但如果10宫主很强+SAV>32，
     事业方面依然有很好的基础

Step 3: 标注突破口
  → 什么时期最可能兑现？（参与星的Dasha时期）
  → 什么领域最可能突破？（参与宫位对应的领域）
```

## 赛道优势合成

```
基于Step 1-3的结果，按四维框架输出：

a) 加持赛道 — 格局命中的领域
   多个Raja Yoga → 管理/权力赛道有先天加持
   Dhana Yoga强 → 财富积累赛道有加持
   AK/AmK力量强 → 事业/灵魂使命赛道有加持
   Gajakesari → 教育/学术赛道有加持

b) 经营赛道 — 格局缺失但可后天发展的领域
   无正式格局但功能性等价物存在（如10宫主强+SAV>32）
   → 说"需要后天经营，关键在[具体宫位/行星]"

c) 财富路径 — 钱从哪条路来
   2宫SAV高 + Dhana Yoga → 工资/产品型收入
   11宫SAV高 + 11宫主强 → 人脉/流量型收入
   8宫强 + VRY → 他人资源/投资/佣金型
   D4交叉 → 不动产/实物资产潜力

d) 兑现窗口 — 格局在哪个大运激活
   参与星的Dasha时期 = 格局兑现窗口
   → "你的[格局名]在[行星]大运（[年份]-[年份]）最可能兑现"

禁止表达：
  ❌ "你的层次是高/中/低"
  ❌ "你命中注定会..."
  ❌ "你的命很差/很好"

正确表达：
  ✅ "你在[领域]赛道有[格局名]加持，
     在[Dasha]时期最可能兑现"
  ✅ "财富主要通过[路径]积累，
     [时期]是关键积累期"
  ✅ "[领域]没有格局加持，
     但[宫位/行星]状况不错，后天经营有空间"
```
