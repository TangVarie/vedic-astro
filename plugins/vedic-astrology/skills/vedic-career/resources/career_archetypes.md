## 一、L10落宫职业场景矩阵

> L10 = 10宫主。它落在哪个宫 = 你"上班的地方"在哪个人生领域。

### L10落1宫（事业=自我）
```
核心模式：自雇/个人品牌/事业与身份高度绑定
典型路径：自由职业、个人IP、创始人、独立顾问
风味变体：
  L10=Sun → 领导者IP（CEO、创始人、公众人物）
  L10=Moon → 照护者IP（育儿博主、心理咨询师、护理）
  L10=Mars → 行动者IP（运动员、外科、工程创业）
  L10=Mercury → 知识者IP（作家、媒体人、教育者、分析师）
  L10=Jupiter → 导师IP（教师、法律、灵性导师、顾问）
  L10=Venus → 美学者IP（设计师、艺术家、时尚、美容）
  L10=Saturn → 建造者IP（建筑、制造、基础设施、长线创业）
⚠️ 注意：L10落1宫但Shadbala弱 → "想自己干但力不从心"，建议先积累再独立
```

### L10落2宫（事业→财务/表达）
```
核心模式：靠嘴/靠钱赚钱。金融、销售、演讲、餐饮、家族生意
典型路径：银行/证券/保险、演讲培训、餐饮、珠宝、家族企业
风味变体：
  L10=Jupiter/Venus → 金融+艺术/教育交叉
  L10=Mercury → 财经写作、金融分析、会计审计
  L10=Saturn → 传统金融、家族长线经营
  L10=Mars → 投资交易（高波动）、军需
⚠️ 2宫同时是Maraka宫 → 事业压力可能反噬健康（长期高压型工作）
```

### L10落3宫（事业→沟通/技能/勇气）
```
核心模式：靠手艺/技能/沟通为生。媒体、自媒体、短途贸易、手工
典型路径：记者、编辑、自媒体运营、销售、短视频、培训师、手工匠人
风味变体：
  L10=Mercury → 写作/媒体/编程/数据（最佳匹配）
  L10=Venus → 设计/摄影/音乐/短视频美学
  L10=Mars → 运动教练、健身自媒体、机械维修
  L10=Moon → 本地社区服务、邻里关系型生意
⚠️ 3宫是upachaya宫 → 越努力越好，但起步可能辛苦
⚠️ 需细分：3宫+6宫/10宫强连=企业技能岗；3宫+11宫强连+6/10弱=纯自由职业
```

### L10落4宫（事业→家/地产/教育）
```
核心模式：Home-based或地产/教育/汽车。事业有"根据地"感
典型路径：房地产、教育(学校/机构)、汽车、农业、政府(本地)、居家办公
风味变体：
  L10=Moon → 教育/育儿/心理（最佳匹配）
  L10=Venus → 室内设计/家居品牌/酒店
  L10=Saturn → 房地产/建筑/政府行政
  L10=Jupiter → 大学教授/学术机构管理
⚠️ L10落4宫=事业很可能在家乡或跟家有关，不一定是"在家工作"
```

### L10落5宫（事业→创意/投机/教导）
```
核心模式：靠脑子/创意/教学赚钱。智力密集型
典型路径：创意产业、投资投机、教育(教学端)、娱乐、政治顾问、研发
风味变体：
  L10=Mercury → 教育科技/编程/游戏设计/量化交易
  L10=Venus → 娱乐/影视/音乐/设计
  L10=Jupiter → 高等教育/哲学/法律教学
  L10=Sun → 政治/娱乐圈/公共领导
  L10=Rahu → 科技创业/投机/非传统创意（元宇宙/AI）
⚠️ 5宫是trikona → 天然有福气加持，但5宫也管投机 → 收入波动性可能大
```

### L10落6宫（事业→服务/竞争/医疗）
```
核心模式：在竞争环境中靠服务/解决问题为生
典型路径：医疗、法律诉讼、军警、人力资源、债务催收、宠物、NGO
风味变体：
  L10=Mars → 外科/军事/竞技体育/执法
  L10=Mercury → 法务/审计/数据纠错/诊断
  L10=Saturn → 劳动密集型管理/工厂/质检
  L10=Jupiter → 医疗教育/公共卫生/社会服务
⚠️ L10落6宫=工作总有"对手/问题要解决"的成分，不会轻松
⚠️ 凶星管L10+落6宫=反而好（凶星在upachaya宫发力）
   吉星管L10+落6宫=Kendradhipati异化，工作消耗快乐
```

### L10落7宫（事业→合伙/外交/贸易）
```
核心模式：事业必须有"另一个人"——合伙人、客户、配偶
典型路径：合伙创业、客户服务、外贸、外交、婚姻咨询、律师(对客型)
风味变体：
  L10=Venus → 奢侈品贸易/婚恋服务/美容（最佳匹配）
  L10=Mercury → 经纪/中介/翻译/国际贸易
  L10=Jupiter → 咨询/教育合伙/法律合伙
  L10=Saturn → 长线合伙企业/政府对外事务
⚠️ L10落7宫=事业跟配偶有关联的概率高（可能一起创业或配偶影响职业方向）
```

### L10落8宫（事业→深度/隐秘/转化）
```
核心模式：在不透明领域深耕。研究、保险、遗产、心理、情报
典型路径：研究型岗位、保险精算、心理治疗、法医、遗产管理、密码学/安全
风味变体：
  L10=Mars → 外科/法医/危机管理/军事情报
  L10=Mercury → 信息安全/数据分析/调查记者
  L10=Saturn → 矿业/地下资源/长线研究
  L10=Rahu → 暗网安全/非常规金融/间谍小说式职业
⚠️ L10落8宫=职业生涯会有"断裂+重建"的模式，不是一条直线
⚠️ 这不是坏位置——只是说事业在公众看不见的地方运作
```

### L10落9宫（事业→哲学/远方/法律）
```
核心模式：事业与信仰/教育/国际/法律深度绑定
典型路径：高等教育、国际机构、法律(立法端)、出版、宗教/灵性、旅游
风味变体：
  L10=Jupiter → 大学教授/法官/灵性导师（最佳匹配）
  L10=Sun → 国际组织领导/政治
  L10=Mercury → 出版/翻译/国际媒体
  L10=Venus → 跨国艺术/国际奢侈品/旅游品牌
⚠️ L10落9宫=可能在海外发展事业或事业有强烈的"使命感"
```

### L10落10宫（事业=事业本身）
```
核心模式：Dig Bala位——事业是人生主旋律，事业心极强
典型路径：看L10是哪颗行星→该行星的天然领域就是职业方向
  Sun在10宫 → 政府/领导/权威
  Moon在10宫 → 公众服务/餐饮/护理/旅游
  Mars在10宫 → 工程/军事/体育/外科（Dig Bala=最强）
  Mercury在10宫 → 商业/会计/IT/媒体
  Jupiter在10宫 → 教育/法律/金融/顾问
  Venus在10宫 → 艺术/娱乐/奢侈品/酒店
  Saturn在10宫 → 制造业/基建/政府/长线企业（Dig Bala=最强）
⚠️ 强配置但也意味着"事业就是全部"——工作生活平衡难
```

### L10落11宫（事业→收入/网络/愿望实现）
```
核心模式：事业直接导向赚钱和人脉变现
典型路径：销售/市场、社交平台、投资/基金、网络型创业、社团组织
风味变体：
  L10=Mercury → 社交媒体/广告/电商（最佳匹配）
  L10=Venus → 网红/KOL/社交型品牌
  L10=Jupiter → 大型组织管理/NGO/教育集团
  L10=Rahu → 科技平台/流量型创业
⚠️ 事业变现路径畅通但需要"人脉"才能转化——孤狼模式不行
```

### L10落12宫（事业→海外/隐退/幕后）
```
核心模式：事业在"看不见的地方"或海外运作
典型路径：海外工作、医院/监狱/寺庙(隔离机构)、幕后研发、灵性行业
风味变体：
  L10=Saturn → 跨国企业驻外/监狱管理/长线海外经营
  L10=Venus → 海外艺术/国际酒店/灵性美学
  L10=Jupiter → 海外教育/灵性教师/慈善
  L10=Moon → 海外护理/移民服务
  L10=Rahu → 跨境电商/移民中介/非常规国际业务
⚠️ L10落12宫≠失业——而是"事业形态不符合常规认知"
⚠️ 花钱赚钱型（先投入后回报）或者事业本身就是一种"消耗"
```

## 二、AmK（事业指标星）原型矩阵

> AmK = Chara Karaka中度数第二高的星。它是AK（灵魂）的"执行官"。
> L10决定"在哪干"，AmK决定"用什么方式干"。

| AmK行星 | 核心手段 | 典型表达 | 与L10组合效应 |
|---------|---------|---------|-------------|
| Sun | 权威/领导/曝光 | 管理层、公务员、政治、品牌代言 | L10+Sun AmK → 权力型事业，需要站在前台 |
| Moon | 感知/照护/公众 | 服务业、医护、餐饮、公共关系 | L10+Moon AmK → 需要"读懂人心"才能出成果 |
| Mars | 行动/竞争/技术 | 工程、军事、外科、运动、创业 | L10+Mars AmK → 事业靠"执行力+冲劲"驱动 |
| Mercury | 沟通/分析/交易 | IT、媒体、教育、金融分析、商务 | L10+Mercury AmK → 事业靠"脑子+嘴"驱动 |
| Jupiter | 智慧/教导/扩张 | 教育、法律、金融、顾问、灵性 | L10+Jupiter AmK → 事业靠"知识储备+信任度"驱动 |
| Venus | 美学/关系/享受 | 艺术、设计、娱乐、奢侈品、外交 | L10+Venus AmK → 事业靠"品味+人际"驱动 |
| Saturn | 纪律/耐力/结构 | 制造、基建、政府、长线投资、审计 | L10+Saturn AmK → 事业靠"苦功夫+时间"驱动，晚成型 |

### AmK状态对事业执行力的影响
```
AmK入旺/入庙 → 执行力强，事业方向明确且易变现
AmK Vargottama → 最佳——事业表里如一，深层品质好
AmK落陷 → 方向正确但执行困难，需后天补强
AmK燃烧 → 执行力被"权威/ego"压制——可能一直给别人打工
AmK逆行 → 执行方式非常规——可能需要"从后门进入"行业
AmK在D10中强势 → 事业执行力有D10背书，确认方向
AmK在D10中弱势 → 事业执行力在深层缺乏支撑，需调整策略
```

## 三、L10 × AmK 合成公式

```
最终职业画像 = L10落宫(场景) + L10是哪颗星(色彩) + AmK(手段) + 强星(放大器)

示例：
  L10=Mercury落11宫 + AmK=Saturn
  = "在收入/网络领域(11宫)，用沟通分析(Mercury)的方式，
    靠长线纪律(Saturn AmK)积累。"
  → 具体：数据分析驱动的长期电商运营 / 社媒矩阵慢慢养号
  → 不是：一夜爆火的网红（Saturn AmK不支持快速变现）

  L10=Venus落5宫 + AmK=Mars
  = "在创意/投机领域(5宫)，用美学(Venus)的方式，
    靠行动力和竞争力(Mars AmK)突围。"
  → 具体：独立游戏开发 / 竞技类设计 / 格斗健身美学品牌
  → 不是：安静的画廊画家（Mars AmK要求冲劲和竞争）
```

## 四、D10验证清单

```
D10在career中不是"另一套系统"——它是D1事业判断的"品质检验"。

D10验证步骤（Phase 1中执行）：
  1. D10 Lagna星座 → 事业的"内在基调"
     与D1 L10对比：一致=方向确认，不一致=有内外落差
  2. D10 10宫主 → 事业成就的"深层引擎"
     强(入旺/入庙) = 成就有深度支撑
     弱(落陷/敌方) = 事业表面风光但根基不稳（D9铁律3同理）
  3. D10中最强星 → 事业的"隐藏王牌"
     如果D10最强星≠D1的L10或AmK → 事业有"第二赛道"潜力
  4. D10中Sun位置 → 权力/认可的深层模式

⚠️ D10可信度阈值：
  时间精度±12分钟以内 → D10正常使用
  时间精度±12-15分钟 → D10仅供参考(⚠️)
  时间精度>±15分钟 → D10不启用
```

## 五、10宫内星（Occupants）速查

> L10定"管家"，10宫内星定"租客"。租客直接给事业上色。

| 10宫内星 | 事业色彩 | 注意事项 |
|---------|---------|---------|
| Sun | 权威/政府/领导力 | 最佳位——Dig Bala，但压力也大 |
| Moon | 公众/流动性/服务 | 工作多变、公众可见度高 |
| Mars | 技术/竞争/高强度 | Dig Bala——技术实力极强但职场冲突也多 |
| Mercury | 沟通/多任务/灵活 | 适合需要多线程处理的工作 |
| Jupiter | 扩张/教育/管理 | 大气但可能"管太宽" |
| Venus | 美学/和谐/品位 | 工作环境好但可能缺乏紧迫感 |
| Saturn | 纪律/长线/苦工 | Dig Bala——最终成就大但过程苦 |
| Rahu | 非常规/科技/国际 | 创新型但不稳定，Dasha期波动大 |
| Ketu | 灵性/技术纯粹/无所谓 | 技术大牛型但不在乎升职加薪 |
| 无星(空宫) | 纯看L10+相位 | 不代表事业差——只是"租客没来，全看管家" |
